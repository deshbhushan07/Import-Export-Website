About AC Global Trade

Why I built this website (add this to your repo description):

I created this website for three reasons:

  1. Online presence: to showcase our import–export capabilities, products, and certifications to potential buyers.
  2. Learning & portfolio: to practice real-world frontend development and maintain a professional portfolio on GitHub.
  3. Documentation: to store product data and company details in a maintainable repository that can be updated and deployed via GitHub Pages.

Mission

To make international trade simple, transparent, and reliable for buyers and suppliers by providing sourcing, logistics, and compliance support.

How to use this repo

  • Edit products in data/products.json.
  • Update site content in the HTML files under the project root and products/.
  • Deploy using GitHub Pages from the repository settings.
